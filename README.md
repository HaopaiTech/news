# news

## 模板

请将以下内容加入到文件顶部（包括 `---`）：

```yml
---
# 新闻标题，必填，推荐与文件名一致
title: 新闻标题
# 新闻日期，必填，务必使用 "" 包裹
date: "2021-07-02"
# 文章图片封面 URL 地址，可选
# 如果为空，则使用 https://source.unsplash.com/800x450/?city
cover: /news/the-birth-of-smartpai.png
# 文章 URL，必填
permalink: the-birth-of-smartpai
# 关键词，必填，按以下格式
keywords:
  - 关键词一
  - 关键词二
  - ……
---
```
